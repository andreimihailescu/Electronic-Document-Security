1) Instalam "Node" din fisierul "node-v6.11.0-x64.msi".

2) Pornim "Command Prompt".

3) Mergem pe urmatorul path: "../Securitatea Documentelor Electronice - Slavila Alexandru Mihailescu Andrei/bin".

4) Rulam urmatoarea comanda in cmd: "node www".

5) Deschidem un browser si mergem pe adresa: "localhost:8081".

6) Incarcam o imagine cu un simplu click pe "Browse" (sunt cateva imgini in folderul "../Securitatea Documentelor Electronice - Slavila Alexandru Mihailescu Andrei/public/images").

7) In primul textarea este imagine in format base64, in cel de al doilea textbox este imaginea criptata cu ajutorul Triple DES.

8) Pentru a vedea rezultatele de pe server trebuie intrat pe "../Securitatea Documentelor Electronice - Slavila Alexandru Mihailescu Andrei/public/download", unde vom gasi trei fisiere unul cu imaginea in format .jpg, unul cu imaginea in format Base64 si unul cu imaginea criptata cu Triple DES.

PS: 
-Fisierul unde s-a facut criptarea este : "../Securitatea Documentelor Electronice - Slavila Alexandru Mihailescu Andrei/public/javascripts/index.js".

-Fisierul unde s-a facut decriptarea este: "../Securitatea Documentelor Electronice - Slavila Alexandru Mihailescu Andrei/APIs/index.js".
